### Samasama
Samasama is a gamified English language acquisition mobile app.  Users raise a Tamagotchi-style sprite from infant to teenager aging the sprite by completing English pronunciation exercises. The learning journey mirrors native language development, progressing from syllable sounds and mouth shapes through nursery rhymes, songs and increasingly complex speech patterns using repetition and memorisation. Voice analysis and facial recognition technology verify pronunciation and articulation, providing objective feedback as users master the sounds and rhythm of a language.

Samasama is a mobile-first language acquisition platform built around pronunciation, intonation and enunciation training.

Frontend
React Native + Expo (TypeScript)

Backend
Fastify (TypeScript)
PostgreSQL 16
Prisma ORM and migrations
Shared API contracts via a monorepo architecture

Voice Analysis
Python FastAPI microservice
Audio processing and pronunciation assessment

Infrastructure
Monorepo structure with shared contracts between frontend and backend

Repo:
https://github.com/dan-griffiths-dev/Samasama
